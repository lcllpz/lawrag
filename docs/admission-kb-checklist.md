# 招生知识库初始化清单

用于将当前系统从法律问答切换为学校招生问答后的首轮数据准备与上线检查。

## 1. 必备资料（建议优先上传）

- 招生章程（当年）
- 招生计划（按省份/批次/专业组）
- 历年录取数据（至少近 3 年，含分数线与位次）
- 专业选科要求说明
- 学费、住宿费、奖助学金政策
- 报名与志愿填报流程说明
- 常见问题 FAQ（学校官方口径）

## 2. 推荐分类（对应前端上传分类）

- `policy`：招生章程
- `plan`：招生计划
- `rule`：录取规则
- `score`：历年录取分数
- `major`：专业与选科要求
- `tuition`：学费与奖助学金
- `application`：报名与志愿填报
- `faq`：常见问题
- `general`：其他

## 3. 文档命名规范（建议）

建议统一命名：

`学校名_年份_省份_资料类型_版本号`

示例：

- `XX大学_2026_浙江_招生章程_v1.pdf`
- `XX大学_2026_浙江_招生计划_v1.xlsx`
- `XX大学_2023-2025_浙江_历年录取位次_v1.pdf`

## 4. 数据质量要求

- 同一政策避免重复上传多个冲突版本
- 明确标注年份，过期资料单独归档
- 尽量保留原始表格/附件中的关键字段（批次、专业组、位次）
- FAQ 答案需与最新章程一致

## 5. 上线前验证项

- 配置检查：`application.yml` 中 Milvus 集合为 `admission_knowledge`
- 提示词检查：`admission_qa.txt`、`query_rewrite.txt`、`safety_check.txt` 已为招生语义
- 抽样提问（至少 10 条）：
  - 能否正确引用来源
  - 是否包含年份/省份/批次等条件
  - 是否避免绝对化承诺（如“保证录取”）
- 兜底回复检查：未命中时是否提示“以官方最新公告为准”

## 6. 首批测试问题（建议）

- “浙江物化生，位次 6200，能报你们学校计算机吗？”
- “2026 年你们学校在江苏本科批投档线大概多少？”
- “不服从调剂会有什么风险？”
- “临床医学专业选科要求是什么？”
- “家庭困难可以申请哪些资助？”
